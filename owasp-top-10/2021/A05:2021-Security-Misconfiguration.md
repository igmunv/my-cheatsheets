# A05: Security Misconfiguration
или **Небезопасная конфигурация**

## Описание
Ситуация, когда конфигурация системы не является безопасной

## Примеры
- Логины и пароли по умолчанию: в базе данных оставили пароль по умолчанию
- Небезопасные права для файлов: можно перезаписать файл, который исполняется пользователем с высокими правами
- Доступ к чувствительным данным: веб-сервер позволяет просматривать то, что не предназначено для пользователя
- Функции, которые не требуются для текущего приложения, оставлены включёнными: могу иметь небезопасные настройки

## Рекомендации по исправлению
- Менять пароли по умолчанию на более безопасные
- Выключать избыточные функции: Сокращения спектра атак
- Регулярно проводить проверку безопасности конфигураций
- Продумать и **придерживаться** политики настроек доступа
